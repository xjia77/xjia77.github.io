Vertex & fragment shader in GLSL

Introduction

This project is a 3D geometry model using vertex and fragment shaders. The purpose of doing this project is to get familiar with the GLSL shading language, the respective roles of vertex shaders and fragment shaders, the exchange of information between them (via variable variables), and the corresponding properties.
