Forms Web Page

2023-01-18

Introduction

This project is a web page containing a user-created form using the Vue framework. It implements the following functionality:

  • Display a form with inputs for each of the above fields
  • Allows the user to complete and submit the form
  • Does not allow the form to be submitted without completing the form
  • Provide feedback after a successful form submission

Running instructions

Locate the form directory, type npm install and npm run serve in terminal, wait for building to complete, and click on the generated localhost link.
